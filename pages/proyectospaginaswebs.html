Proceso UX

Proceso UX EFE trenes de Chile

El proceso UX de Efe Metros se enfocó en comprender y mejorar la experiencia del usuario mediante herramientas estratégicas. Se comenzó con un mapa de trayectoria y un Lean UX Canvas para definir objetivos y detectar oportunidades. Luego, se organizó la información usando Card Sorting y Tree Testing (Dendodrama), y se evaluó la percepción del producto con la matriz FVD. Se diseñaron los flujos con wireflows, y se alineó el propósito del proyecto con el Círculo de Oro y la Matriz de Voz de Marca para lograr una comunicación clara y coherente.

Proceso UI

Proceso de Kit ui y rediseño de EFE trenes de Chile

Este proyecto consistió en el rediseño visual de la interfaz digital de EFE Trenes de Chile, desarrollando un UI Kit completo con componentes reutilizables, estilos tipográficos, iconografía, paleta de colores y botones coherentes con la identidad de marca. Se mejoró la jerarquía visual y la accesibilidad, garantizando una experiencia más clara y amigable para los usuarios. El sistema de diseño creado permite mantener consistencia en futuras implementaciones y facilita el trabajo colaborativo entre diseñadores y desarrolladores.
